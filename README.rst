MooQuant 是一个基于 pyalgotrade 衍生而来的完全支持 python3.x 版本 的支持国内A股的量化交易框架。


* 协议: MIT license
* 文档: https://mooquant.readthedocs.io.


功能
--------

* 升级 tornado 版本到 5.1
* 添加 tushare 做为数据源
* 添加 mootodx 作为数据源
